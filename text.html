Thai and Chinese are special, if the website needs to provide these languages, we will add new fonts.Please note that all text needs to declare the font style, and will not use the system or browser default fonts.

Text Common Color

Text intended to be read has three regular gray colors, and we try not to use other grays. Exceptions: There are times when the text will use the brand color and when the background has color and images will use white.

color: rgba(32, 35,37, 1);

color: rgba(32, 35,37, 0.6);

color: rgba(117, 117,117, 0.3);

Dark gray

Medium gray

Light gray

The dark gray color is mainly used in headlines. This color is strong and eye-catching, allowing users to quickly capture key information such as headlines, topics, and sections. Also, if some functional UI text itself is too small in size, we worry that users will ignore them, so we also use this color to enhance the presence of the text.

Medium gray is mainly used for body text, paragraphs that require long reading time. This color is a bit brighter than the one usually used for body text, making the overall style of our website lighter. If it appears in the list, it usually means that the paragraph is not selected or activated. If it appears elsewhere, it usually means that the text is not interactive.

The light gray color is mainly used for placeholders and appears where the user needs to enter text. This color is so light that it is acceptable to be ignored by the user.
